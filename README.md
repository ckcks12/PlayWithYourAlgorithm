## Hotfix
I put two `.ocx` files into `hotfix` folder so move them to `C:\Windows\SysWOW64` (for x86, `C:\Windows\System32`) and run below command with administrator permission.
```
regsvr32 MSCOMCTL.ocx
regsvr32 comdlg32.ocx
```
